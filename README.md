# 🤖 Dorami AI

Dorami AI is a real-time voice assistant built using Python that can understand voice commands, perform tasks, and respond with natural speech.

## 🚀 Features
- 🎤 Voice recognition (SpeechRecognition)
- 🔊 AI voice responses (Edge TTS)
- 🧠 Command processing engine
- 🌐 Web search + Wikipedia summary
- 🎵 YouTube + Spotify control
- 🖥 System automation (apps)

## 🛠 Tech Stack
- Python
- SpeechRecognition
- Edge TTS
- PyWhatKit
- Wikipedia API

git clone https://github.com/drithirn/dorami-ai.git
